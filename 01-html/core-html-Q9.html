Text Formatting Tags

  1. < b >
    • Bold text
    • Use sparingly to highlight important text.
  2. < strong >
    • Important text
    • Use for text that is essential to the meaning of the document.
  3. < i >
    • Italic text
    • Use for emphasis or to indicate a secondary thought.
  4. < em >
    • Emphasized text
    • Use for text that is important but not essential to the meaning of the document.
  5. < u >
    • Underlined text
    • Use sparingly to highlight important text.
  6. < mark >
    • Marked text
    • Use to highlight text that has been searched for or that is otherwise noteworthy.
  7. < small >
    • Smaller text
    • Use for text that is secondary or supplementary.
  8. < del >
    • Deleted text
    • Use to indicate text that has been removed from the document.
  9. < ins >
    • Inserted text
    • Use to indicate text that has been added to the document
  10. < sub >
    • Subscript text
    • Use for text that should be displayed below the baseline.
  11. < sup >
    • Superscript text
    • Use for text that should be displayed above the baseline.

Full stack web development tech stack

HTML
HTML is the language used to create web pages. It stands for HyperText Markup Language.
CSS
CSS is used to style HTML elements. It stands for Cascading Style Sheets.
JavaScript
JavaScript is used to add interactivity to web pages. It is a programming language.
Node.js
Node.js is a JavaScript runtime environment. It is used to create server-side applications.
Express.js
Express.js is a web application framework built on top of Node.js. It is used to create scalable and reliable web applications.
MongoDB
MongoDB is a NoSQL database. It is a document-oriented database that is used to store data in JSON format.
MySQL
MySQL is a relational database. It is a popular database that is used to store data in tables.
Apache
Apache is a web server. It is used to serve web pages to users.
Git
Git is a version control system. It is used to track changes to code.
GitHub
GitHub is a code hosting platform. It is used to store and share code.
